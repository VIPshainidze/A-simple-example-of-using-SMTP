# A-simple-example-of-using-SMTP

­ЪЊб­ЪЊб­ЪЊб рЃарЃљ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ ­ЪЊб­ЪЊб­ЪЊб
­ЪЊб Python 3-рЃўрЃА smtplib рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА SMTP рЃърЃарЃЮрЃбрЃЮрЃЎрЃЮрЃџрЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃњрЃќрЃћрЃЉрЃў рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃЊрЃљрЃАрЃЏрЃърЃљрЃЏрЃЋрЃЮрЃА рЃарЃўрЃАрЃЎрЃћрЃЉрЃўрЃА рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃўрЃќрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃЎрЃЋрЃћрЃБрЃџрЃў рЃњрЃќрЃћрЃЉрЃўрЃЌ.  SMTP: рЃћрЃА рЃљрЃарЃўрЃА рЃърЃарЃЮрЃбрЃЮрЃЎрЃЮрЃџрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃћрЃџрЃўрЃљ рЃћрЃџ.рЃцрЃЮрЃАрЃбрЃћрЃЉрЃўрЃА рЃЊрЃљрЃњрЃќрЃљрЃЋрЃюрЃљрЃАрЃљ рЃЊрЃљ рЃцрЃЮрЃАрЃбрЃўрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃћрЃџ.рЃцрЃЮрЃАрЃбрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃерЃарЃБрЃбрЃўрЃќрЃўрЃарЃћрЃЉрЃљрЃќрЃћ. ­ЪЉеРђЇ­Ъњ╗­ЪЉеРђЇ­ЪјЊ
